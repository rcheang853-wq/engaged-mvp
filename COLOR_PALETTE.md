## Usage

### Member Color Assignment
Rotate through: Blue → Green → Orange → Red → Purple → Pink

**Example:**
- Member 1: #3B82F6 (blue)
- Member 2: #10B981 (green)
- Member 3: #F59E0B (orange)
- Member 4: #EF4444 (red)
- Member 5: #8B5CF6 (purple)
- Member 6: #EC4899 (pink)
- Member 7: #3B82F6 (back to blue)

### Event Colors
Events inherit the color of the member who created them.
Users can also pick custom colors from the palette.

### Backgrounds
- Main app: #F9FAFB
- Cards: #FFFFFF (white)
- Input fields: #F3F4F6

### Text
- Headings: #111827
- Body: #374151
- Secondary: #6B7280
- Tertiary/hints: #9CA3AF

### Buttons
- Primary: #3B82F6 (text: white)
- Secondary: #F3F4F6 (text: #374151)
- Danger: #EF4444 (text: white)
- Success: #10B981 (text: white)
