# How to Use Pencil Desktop App for This Project

## Step 1: Open Pencil Desktop App
Launch the Pencil app on your computer.

## Step 2: Create New Document
- Click **"Create New Document"**
- Choose **"Mobile GUI Prototyping"** template
- Set canvas size to **375 x 812** (iPhone standard)

## Step 3: Read the Design Brief
Open `DESIGN_BRIEF.md` to see:
- All 5 screens to design
- Layout specifications
- Design system rules

## Step 4: Reference the Colors
Open `COLOR_PALETTE.md` and use those exact hex codes in Pencil.

## Step 5: Design Each Screen
Create 5 separate pages in your Pencil document:
1. Calendar List
2. Monthly Calendar View
3. Event Detail
4. Create Event
5. Join Calendar

Use the design brief as your guide for each screen.

## Step 6: Use These UI Elements
From Pencil's stencil library, use:
- **Rectangles** → cards, buttons
- **Text** → headings, labels
- **Circles** → avatars, color dots
- **Icons** → arrows, plus, settings (built-in icon stencil)

## Step 7: Export Your Designs
When done:
1. File → Export → PNG
2. Save each page separately:
   - `1-calendar-list.png`
   - `2-monthly-view.png`
   - `3-event-detail.png`
   - `4-create-event.png`
   - `5-join-calendar.png`

## Step 8: Send Back
Share the 5 PNG files and I'll code them into the real app!

## Quick Tips

### Rounded Corners
- Cards: 16px radius
- Buttons: 12px radius
- Avatars: 50% (full circle)

### Spacing
- Between cards: 12px
- Inside cards (padding): 16px
- Between UI elements: 8-12px

### Font Sizes
- Page title: 20px, bold
- Card heading: 16px, bold
- Body text: 14px
- Small labels: 12px
- Tiny text: 10px

### Shadows
Use subtle gray shadows on cards (2-4px blur, 10-20% opacity black)
